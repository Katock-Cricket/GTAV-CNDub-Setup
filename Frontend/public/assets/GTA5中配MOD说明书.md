## 介绍

我们为GTA5制作了汉语配音模组，内容如下：

| 类型     | 内容                                                         |
| -------- | ------------------------------------------------------------ |
| 主角配音 | [必选]真人配音，常见角色包括三主角在大世界中的触发式语音     |
| 剧情配音 | [必选]真人配音，部分前期剧情任务的语音、大世界事件           |

## 卸载方法

> **[注意]**
>
> 中配模组包含这些组件：`mod本体`、`mod加载器.asi`、`asi加载器.dll`，如果你之前安装过别的模组，那你大概率已经有`mod加载器.asi`和`asi加载器.dll`了，一般是`OpenIV.asi`、`OpenRPF.asi`、`dinput8.dll`、`xinput1_4.dll`，你的其他MOD可能依赖这些文件。如果要恢复线上功能，请删除asi加载器，即`dinput8.dll`、`xinput1_4.dll`等文件。
>

| 传承版(老版)                                                 | 增强版                                                       |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 如果使用安装器一键卸载(直接删除mods文件夹中与中配相关的RPF文件)，请注意防止其他处于同一RPF文件内的已安装模组被一起误删。如果要精细地删除中配内容，请使用[OpenIV](https://www.gta5-mods.com/tools/openiv)操作。过于详细的方法不再赘述。 | 如果直接删除mods文件夹中与中配相关的RPF文件，请注意防止其他处于同一RPF文件内的已安装模组被一起误删。如果要精细地删除中配内容，请使用**大于等于dev48版**的[CodeWalker](https://discord.com/channels/329138547833700352/351357358460370944)操作，旧版CodeWalker与OpenIV都不兼容增强版。过于详细的方法不再赘述。 |

## 常见问题

1. **安装后没有起效果，开屏logo也不是中配logo**：检查asi loader是否正确安装，可以使用本模组`pre_install`文件夹中提供的，传承版使用`dinput8.dll`，增强版使用`xinput1_4.dll`，并删除原来安装的asi loader。
2. **三主角和自机还有一些语音没有中配**：顾不上弄了，鸽了半年也没憋出来，主要演员们都没时间，以后随缘更新了；
3. **“掌上明珠”以后的主线没有中配**：原因同上；
4. **使用安装器对增强版安装后不生效或无法启动**：如果你之前没有安装过任何MOD，游戏版本高于上面通过测试的版本，则可能是游戏更新导致的，与本MOD无关，请手动更新`OpenRPF.asi`、`xinput1_4.dll`等；如果你本身就有asi loader，请检查是不是安装重了，asi loader一般命名为`xinput1_4.dll`、`dsound.dll`等，只保留一个可以用的就行；
5. **安装器无法启动**：安装器有两项依赖，见上文。如果你电脑比较老，可能没有，启动安装器时会弹窗提示，请先到`pre_install`中运行对应的安装程序；
6. **BattlEye相关报错**：安装任何模组后，GTA5可能因BattlEye(反作弊程序)检测到额外加载项而无法正常启动，请尝试在R星启动器中禁用BE反作弊，或在exe的“启动选项”的里添加-nobattleye参数。禁用后无法使用线上功能，也就是说模组只能线下玩，不能与OL共存；
7. **ScriptHookV相关报错**：中配MOD不依赖于ScriptHookV，任何与ScriptHookV有关的报错均与中配MOD无关；
8. **增强版主线任务字幕不匹配**：由于增强版文件格式是Gen9，目前只有CodeWalker+OpenRPF提供了转换+挂载的功能，但还不完整，导致字幕依然不能顺利加载，我能力范围内做不到这点，所以增强版不能安装配套字幕，安装器也会自动跳过这项。
9. **电视节目没有配音/加载电视节目时闪退ERR_GEN_ZLIB_2/好友请求和掌上明珠闪退**：较新版本的GTA5可能无法替换电视节目了，电视节目位于mods/x64d.dpf，可选择性安装或删除。
